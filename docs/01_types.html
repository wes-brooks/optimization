Types of optimization problems

There are two broad kinds of optimization problems that you are likely to encounter: those with continuous parameters, and those with discrete parameters.

Continuous parameters

In the case of continuous parameters (like the gravity example before), the basic task is to where the derivative of loss with respect to the parameters is zero. Most (all?) algorithms will find a local minimum, with no guarantee that it finds the global minimum. There are two basic categories of optimizer for continuous parameters: closed form and iterative. A closed form optimizer is a special solution to a specific problem, and generally canâ€™t be applied to a different problem. So, most (all?) generic optimizers are iterative: they progress toward the solution in repeated steps until the steps get small enough that we say the algorithm has converged.

Next, we can break optimization algorithms into categories based on how much information we have about the derivatives of the function w.r.t. the parameters. The most common categories are to have: 0. No knowledge of the derivatives. 1. Known first derivative. 2. Known first and second derivatives.

Here is an example:

gd_iterate = function(gamma, x_last) {
  finished = FALSE
  while (!finished) {
    # x_last = tail(path, 1)
    
    gamma = gamma / 0.8 * (-sign(fww1(x_last)))
    trial = fww(x_last + gamma)
    if (trial > fww(x_last) + 0.2*gamma*fww1(x_last)) {
      
      gamma = gamma * 0.8^2
    } else {
      x_next = x_last + gamma
      # path = c(path, x_next)
      finished = TRUE
    }
  }
  
  lines( c(x_last, x_next), c(fww(x_last), fww(x_last) + gamma*fww1(x_last)), lty=2)
  points(x_last, fww(x_last), col='red', pch=1)
  
  list("gamma" = gamma, "x_new" = x_next)
}

path = c(2.5)
gamma = 0.2

# iterate this part to get the solution:
result = gd_iterate(gamma, tail(path, 1))
